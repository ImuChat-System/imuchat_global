# 🎛 3️⃣ Architecture recommandée

Tu as déjà dans ton résumé :

> Ambiances sonores : player global, playlists thématiques, crossfade & mute automatique sur événement 

Donc voici l’architecture logique :

### 🎧 Core Music Engine

* Mini-lecteur global (dockable)
* Crossfade automatique
* Auto-pause si appel
* Volume indépendant des notifs
* Mode “ambiance chat” (par conversation)

---

# 🌍 4️⃣ Intégration des grandes plateformes

Tu as raison : **API + mini-apps via Store**

Voici les plateformes pertinentes :

* Spotify
* Apple Music
* YouTube Music
* Deezer
* SoundCloud

### 2 niveaux d’intégration possibles :

### 🔹 Niveau 1 (simple)

* Affichage du titre en cours
* Bouton “ouvrir dans l’app”
* Partage dans le chat

### 🔹 Niveau 2 (avancé)

* Lecture intégrée
* Playlists synchronisées
* Watch Party audio
* “Écoute ensemble”
