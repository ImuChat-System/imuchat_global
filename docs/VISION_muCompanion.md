ImuCompanion est un **moteur natif d’avatars intelligents incarnés**, intégré au cœur d’ImuChat, combinant :

* Avatar animé (2D/3D)

# 2️⃣ Rendering Engine (Live2D + RN)

## 🎭 Objectif

Rendre un avatar animé 2D performant mobile-first.

---

## Option recommandée

### 📱 Mobile (iOS / Android)

* React Native + custom native module
* Intégration Live2D via WebGL view ou moteur natif
* Surface de rendu isolée (GLView)

Avec Expo :

* Utiliser expo-gl
* Créer un bridge natif si nécessaire

---

## Architecture Rendering

```
ImuAvatarView.tsx
 ├── GL Surface
 ├── Model Loader
 ├── Animation Controller
 ├── Expression Controller
 └── LipSync Controller
```

---

## 🎬 Composants clés

### 1. ModelLoader

* Charge .model3.json
* Précharge textures
* Cache mémoire

### 2. AnimationController

* Idle loops
* Trigger animations
* Context-based motion

### 3. ExpressionController

* Paramètres Live2D :

  * ParamEyeOpen
  * ParamMouthOpen
  * ParamAngleX/Y
  * ParamSmile

### 4. LipSync

* Analyse amplitude audio TTS
* Map amplitude → ParamMouthOpen

---

# 3️⃣ Behaviour Engine (logique émotionnelle)

Ce n’est PAS l’IA.
C’est un moteur d’état.

## 🎛 Basé sur une FSM (Finite State Machine)

États :

* Idle
* Listening
* Thinking
* Speaking
* Reacting
* FocusMode
* LowPowerMode

Transitions déclenchées par :

* Message reçu
* IA répond
* Utilisateur parle
* Mode réunion activé
* Batterie faible

---

## Structure TypeScript

```ts
type CompanionState =
  | "IDLE"
  | "LISTENING"
  | "THINKING"
  | "SPEAKING"
  | "REACTING"
  | "FOCUS"
  | "LOW_POWER";
```

Un reducer global gère transitions.

# 4️⃣ AI Pipeline détaillé

## 🧠 Architecture IA complète

```
User Input
   │
   ▼

   │
   ▼

   ▼

   ▼
Post Processing

   ├── Animation Trigger
```

## 4.3 LLM Invocation

## 4.4 Post Processing

Très important.

On renvoie :

```json

  "emotion": "calm_confident",

  "tts": true,
  "tool_calls": []

```

CRITIQUE.

## 🎯 Objectifs

* < 120MB RAM total app mid-device

## Techniques d’optimisation

### 1️⃣ FPS adaptatif

* 30fps normal
* 15fps low battery
* Pause animation hors écran

### 2️⃣ Lazy Loading

* Charger modèle seulement si activé
* Décharger textures si inactif

### 3️⃣ Animation Throttling

* Désactiver micro-mouvements invisibles

### 4️⃣ Memory Management

* Nettoyage textures GL
* Garbage collection maîtrisée
