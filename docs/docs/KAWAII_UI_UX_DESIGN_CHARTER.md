# 🌸 ImuChat — Charte "Kawaii UI/UX Design"

## 📅 Date de création : 7 octobre 2025
**Statut** : 📋 PLANIFIÉ  
**Priorité** : 🎨 HAUTE (Impact UX majeur)  
**Phase** : Phase 3 - Design System & Direction Artistique  

---

## 🎯 Vision & Objectif

Transformer **ImuChat** en une expérience sociale **mignonne, kawaii et émotionnellement plaisante**, créant un univers numérique chaleureux où les utilisateurs se sentent accompagnés et bienvenus.

**Objectif principal** : Créer une application sociale chaleureuse, positive et émotionnellement engageante, inspirée du kawaii japonais et du confort digital.

**Valeurs visuelles** :
- 💕 Douceur
- ✨ Légèreté  
- 🌸 Bienveillance
- 🌙 Rêve
- 👥 Amitié

**Ambiance globale** : Un univers calme, pastel et magique, dans lequel l'utilisateur se sent comme dans un cocon numérique.

## 🎨 1. Direction Artistique & Esthétique Générale

### Palette de Couleurs Pastel
**Couleurs principales** :
- Rose pêche (`#FFB7D5`)
- Bleu ciel (`#A6D8FF`)
- Lavande (`#C7A6FF`)
- Crème (`#FFF7FA`)
- Menthe (`#AEE6CF`)
- Blanc nacré (`#FFEAF4`)
- Lilas (`#B994FF`)

### Dégradés Doux
- ✅ Transitions légères entre deux tons pastels
- ✅ Exemple : rose → lavande, menthe → bleu ciel
- ✅ Éviter les transitions brusques

### Icônes & Formes
- **Style** : Arrondies et joyeuses
- **Inspiration** : Line Friends, Sanrio, Pusheen
- **Taille** : 20–32px, espacées et aérées
- **Décorations** : Petites étoiles (✨), cœurs (💖), fleurs (🌸), nuages (☁️)

### Typographie Douce
**Polices officielles** :

| Usage | Police | Alternative |
|-------|--------|-------------|
| Titres kawaii | Poppins Bold / Nunito ExtraBold | Quicksand |
| Corps de texte | Noto Sans JP Regular | Inter Regular |
| Éléments japonais | Noto Sans JP / Rounded M+ 1c | Kosugi Maru |
| Boutons & labels | Nunito SemiBold | Comic Neue |

**Caractéristiques** : Arrondies, sans empattement, lecture fluide et amicale

### Émojis Intégrés
- Dans les titres : "Bienvenue 💕"
- Dans les notifications : "Nouveau message 💌"
- Dans les boutons : "Envoyer 🌸"
- Icons : 💌, 🌸, ✨, 💕, 🧋, 🎀

### Micro-Détails Kawaii
- ✅ Stickers animés
- ✅ Transitions nuageuses
- ✅ Petits visages souriants sur certains éléments UI
- ✅ Bordures fines en dégradé (rose → blanc)

## 🧸 2. Expérience Utilisateur Émotionnelle

### Animations d'Accueil
- **Mascotte** : Imu-chan 🩷 (entité lumineuse ou esprit numérique)
- **Message** : "Okaeri~ 💕" (Bon retour)
- **Animation** : Tintement magique + apparition douce

### Réactions Expressives
**Envoi de message** :
- Message flotte légèrement
- Rebondit au-dessus du chat
- Émet une petite étincelle ✨
- Son : "pop!" (bulle)

**Interactions** :
- Cœurs qui s'envolent après un "like"
- Avatars qui clignent ou sourient selon contexte
- Boutons légèrement rebondissants

### Sons Doux et Cristallins

| Événement | Son | Description |
|-----------|-----|-------------|
| App ouverture | "pling~ 🌸" | Tintement magique |
| Message envoyé | "pop!" | Bruit de bulle |
| Notification | "ding~" | Carillon doux |
| Mascotte | "Hai~ 💕" | Voix mini-robot kawaii |
| Succès | "Yatta~ 🎉" | Jingle court |

🔇 **Règle d'or** : Tous les sons doivent être cristallins, doux et **non répétitifs**.

### Feedbacks Positifs Constants
**Messages d'encouragement** :
- "Yatta ! 🎉"
- "Tu es génial(e) aujourd'hui 💕"
- "Petit moment douceur ☁️"
- "Bienvenue à la maison 💕"
- "Ton espace est prêt 🌸"
- "Tu brilles aujourd'hui ✨"
- "Bravo pour ta régularité 🌷"

### Transitions Naturelles
**Types** :
- Fondu doux
- Glissement fluide
- Effet nuage
- Rebond élastique
- **Jamais** d'animation brutale

### Écrans "Pause" Apaisants
- Fond animé avec nuages flottants
- Musique douce optionnelle (lo-fi)
- Message : "Petit moment douceur ☁️"
- Activation après 5 min d'inactivité

## 🧠 5. Psychologie du Kawaii

### Provoquer l'Attachement Émotionnel
**Méthodes** :
- Présence constante de la mascotte Imu-chan
- Avatar personnalisable qui "grandit" avec l'utilisateur
- Compagnon virtuel qui réagit aux émotions
- Historique émotionnel visualisé (graphique de bonheur)

### Encourager la Bienveillance
**Systèmes** :
- Filtres anti-toxicité avec messages doux ("Soyons gentils 💕")
- Réponses automatiques apaisantes
- Émojis calmants (☁️, 🌸, 💙)
- Mode "Zen" pour conversations tendues
- Rappels de respiration ("Petite pause respiration 🧘‍♀️")

### Utiliser la Surprise Positive
**Easter Eggs** :
- Petits messages secrets aléatoires
- Récompenses mignonnes ("Tu as gagné une fleur 🌼")
- Événements surprise (pluie de sakura, arc-en-ciel)
- Cadeaux virtuels quotidiens
- Animations cachées au long-press

### Favoriser le Lien Humain
**Features** :
- Réactions vocales légères (optionnel)
- Encouragements personnalisés
- Célébration des anniversaires d'amitié
- Messages de groupe illustrés
- Moments partagés (photos, musiques)

### Gamification Douce
**Système de progression** :
- Badges adorables (🌷, 🌙, ⭐, 🎀)
- Progression symbolisée par :
  - Fleurs qui poussent
  - Lunes qui se remplissent
  - Cœurs qui brillent
- **Pas de compétition** : seulement célébration personnelle
- Récompenses visuelles (stickers, thèmes, décorations)

## 🧩 6. Interface et Micro-Interactions

### Composants UI Principaux

#### Boutons
**Style** :
- Larges et arrondis (border-radius: 16–24px)
- Ombre rose douce (`box-shadow: 0 4px 12px rgba(255, 183, 213, 0.3)`)
- Icône à gauche
- Animation rebond au clic
- Effet gradient subtil

**Code exemple** :
```tsx
<Button 
  style={{
    borderRadius: 20,
    padding: '12px 24px',
    boxShadow: '0 4px 12px rgba(255, 183, 213, 0.3)',
    background: 'linear-gradient(135deg, #FF93B3 0%, #FFB7D5 100%)'
  }}
  onPress={() => /* animation rebond */}
>
  <Icon name="send" size={20} /> Envoyer 🌸
</Button>
```

#### Input / Chatbox
**Style** :
- Coins très arrondis (border-radius: 20px)
- Fond légèrement translucide (`rgba(255, 255, 255, 0.8)`)
- Bordure fine rose (`1px solid #FFB7D5`)
- Placeholder kawaii ("Écris ici ✨")

#### Avatars
**Style** :
- Forme circulaire parfaite
- Halo lumineux autour (glow effect)
- Border 2px blanc
- Animation pulse légère
- Badge de statut mignon (💚 en ligne, 🌙 absent)

#### Cartes (Cards)
**Style** :
- Texture papier ou verre dépoli (`backdrop-filter: blur(10px)`)
- Soft shadow : `0 8px 24px rgba(0, 0, 0, 0.08)`
- Border-radius : 16–24px
- Padding généreux (16–20px)
- Hover : légère élévation

#### Popups / Modals
**Style** :
- Flottants avec fond flou
- Émojis dans le titre
- Animation d'apparition : rebond élastique
- Boutons arrondis
- Message court et bienveillant

#### Switches / Toggles
**Style** :
- Animation coulissante fluide
- Effet rebond à la fin
- Couleurs pastel (off: gris clair, on: rose/menthe)
- Petite icône dans le bouton (☀️/🌙 pour day/night)

#### Tabs
**Style** :
- Icônes kawaii
- Effet de sélection scintillant (✨)
- Animation glissement doux
- Indicateur coloré sous l'onglet actif

### Animations et Effets

#### Apparitions Douces
**Types** :
- Fondu (fade-in)
- Rebond élastique (spring)
- Glissement depuis le bas (slide-up)
- Rotation légère (spin 360°)

**Timing** :
- Duration : 300–500ms
- Easing : `cubic-bezier(0.68, -0.55, 0.265, 1.55)` (elastic)

#### Transitions de Page
**Effets** :
- Effet nuage (dissolve doux)
- Glissement latéral (slide)
- Fade croisé (cross-fade)
- Zoom avant/arrière subtil

#### Interactions Spéciales
**"Like" / Réaction** :
- Cœurs qui s'envolent (5–7 cœurs)
- Particules scintillantes
- Son "pling~"
- Animation 800ms

**Message envoyé** :
- Message flotte 2–3px au-dessus
- Léger rebond (bounce)
- Étincelle ✨ au coin
- Son "pop!"

**Avatar hover** :
- Clignement d'œil
- Sourire qui s'agrandit
- Halo qui pulse

#### Chargement (Loading)
**Animation Imu-chan** :
- Mascotte tournoyant dans un nuage
- Message : "Chargement en douceur ☁️"
- Progression douce (pas de spinner brutal)
- Couleurs pastel animées

**Code concept** :
```tsx
<LottieView
  source={require('./assets/imu-loading.json')}
  autoPlay
  loop
  style={{ width: 120, height: 120 }}
/>
```

## 🧱 10. Cohérence Technique

### Architecture Frontend

#### React Native + Expo
**Structure** :
```
/src
  /theme
    theme.ts          # Palette centralisée
    kawaii-themes.ts  # Tous les thèmes kawaii
  /assets
    /themes
      /kawaii
        /icons        # Icônes SVG
        /sounds       # Sons cristallins
        /mascottes    # Imu-chan Lottie
        /decorations  # Pétales, nuages, etc.
  /components
    /kawaii
      Button.tsx
      Card.tsx
      Avatar.tsx
      Modal.tsx
```

**Gestion des thèmes** :
```typescript
// theme.ts
import { configureFonts, MD3LightTheme } from 'react-native-paper';

export const KawaiiTheme = {
  ...MD3LightTheme,
  colors: {
    primary: '#FF93B3',
    secondary: '#AEE6CF',
    background: '#FFEAF4',
    surface: '#FFF7FA',
    // ...
  },
  fonts: configureFonts({
    config: {
      fontFamily: 'Nunito',
    }
  }),
  roundness: 20,
};
```

#### Next.js (Web)
**Structure** :
```
/src
  /styles
    theme.config.ts   # Config Tailwind custom
    kawaii.css        # Classes kawaii custom
  /public
    /assets
```

#### Gestion (React Native)
```tsx
import { Audio } from 'expo-av';

const playSound = async (soundFile: string) => {
  const { sound } = await Audio.Sound.createAsync(
    require(`./assets/sounds/${soundFile}`)
  );
  await sound.playAsync();
};

// Usage
<Button onPress={() => playSound('ui/button-click.mp3')}>
  Envoyer 🌸
</Button>
```

#### Gestion (Next.js)
```tsx
import useSound from 'use-sound';

const [playClick] = useSound('/sounds/button-click.mp3');

<button onClick={playClick}>
  Envoyer 🌸
</button>
```

### Performance & Optimisation

#### Images
- **Format** : WebP (web), optimisé PNG (mobile)
- **Résolutions** : @1x, @2x, @3x
- **Lazy loading** : Charger décorations après contenu principal
- **Sprites** : Combiner petites icônes

#### Animations
- **Limiter simultanées** : Max 3-4 animations en même temps
- **useNativeDriver** : Pour animations RN (transform, opacity)
- **RequestAnimationFrame** : Pour animations custom
- **Throttle/Debounce** : Pour interactions fréquentes

#### Musique/Sons
- **Précharger** : Sons UI critiques
- **Stream** : Musique d'ambiance
- **Compression** : MP3 128kbps suffisant
- **Mute option** : Respecter préférence utilisateur
