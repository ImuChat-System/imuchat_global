        Quelques dossiers models de Live2D pour les tests et les exemples d'utilisation de la bibliothèque Live2D Cubism SDK for Web. Ces modèles peuvent être utilisés pour créer des animations et des interactions dans des projets web en utilisant Live2D. Chaque dossier contient les fichiers nécessaires pour charger et animer le modèle correspondant.
