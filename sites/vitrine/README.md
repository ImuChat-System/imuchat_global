## ✨ Fonctionnalités

- 🌍 **Multilingue** - Support FR, EN, DE, ES, JA (i18n avec next-intl)
- 🎨 **Design moderne** - Gradient premium, animations fluides, responsive
- 📱 **Mobile-first** - Optimisé pour tous les devices
- ⚡ **Performance** - Next.js 14 avec App Router
- 🔍 **SEO-ready** - Meta tags, structure sémantique
- ♿ **Accessible** - Conforme WCAG 2.1

## 🚀 Démarrage Rapide

### Prérequis

- Node.js 20+
- pnpm 8+

### Installation

```bash
# Installer les dépendances
pnpm install

# Lancer le serveur de développement
pnpm dev

# Ouvrir http://localhost:3000
```

### Build Production

```bash
# Build
pnpm build

# Lancer en production
pnpm start
```
