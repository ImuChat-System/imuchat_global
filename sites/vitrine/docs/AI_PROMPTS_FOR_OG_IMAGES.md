# Prompts IA pour les Images Open Graph - ImuChat

Ce document contient tous les prompts optimisés pour générer les images Open Graph avec Midjourney ou DALL-E 3.

## 📋 Instructions générales

### Pour Midjourney

- Ajouter `--ar 1.91:1` pour le ratio Open Graph
- Ajouter `--style raw` pour un rendu plus réaliste
- Ajouter `--v 6` pour la dernière version

### Pour DALL-E 3

- Spécifier "1792x1024 pixels" (sera recadré en 1200×630)
- Bien préciser "no text" si vous voulez ajouter le texte manuellement après
- Utiliser "photorealistic" ou "modern digital illustration" selon le style

### Palette de couleurs à mentionner

```
Primary: vibrant purple #8B5CF6
Secondary: hot pink #EC4899  
Dark: slate navy #0F172A
Background: gradient from purple to pink
```

## 🎯 Workflow recommandé

### Étape 1 : Générer les images

1. **Commencer par l'image par défaut** (`og-image-default.png`)
   - Établir l'identité visuelle de base
   - Définir la palette et le style

2. **Créer les icons PWA** (`icon-192x192.png` et `icon-512x512.png`)
   - Logo simple et reconnaissable
   - Fonctionne à toutes les tailles

3. **Générer les images prioritaires** :
   - `og-image-home.png` (page principale)
   - `og-image-partners.png` (PEPITE)
   - `og-image-news.png` (timeline)

4. **Compléter le reste** des images OG

### Étape 2 : Post-production

Après génération IA, ouvrir dans Figma ou Photoshop pour :

1. **Ajouter le texte** :
   - Titre principal (60-80px, Inter Bold)
   - Sous-titre (32-40px, Inter Regular)
   - Logo ImuChat si nécessaire

2. **Recadrer aux dimensions exactes** :
   - 1200×630 pour les OG images
   - 192×192 et 512×512 pour les icons

3. **Optimiser** :
   - Exporter en PNG de qualité
   - Compresser avec TinyPNG ou ImageOptim
   - Viser < 300 KB par image OG

4. **Zones de sécurité** :
   - Garder 80px de marge sur tous les côtés
   - Éviter le texte important près des bords

### Étape 3 : Template de texte Figma

Créer un template avec calques de texte réutilisables :

```
[Structure Figma]
📐 Template OG 1200×630
  ├── 🔒 Zone de sécurité (guides)
  ├── 🖼️ Image générée IA (locked)
  ├── 📝 Overlay sombre (optionnel, 20% opacity)
  ├── 📍 Logo ImuChat (coin haut gauche)
  ├── 📄 Titre (Inter Bold 64px, blanc)
  ├── 📄 Sous-titre (Inter Regular 36px, slate-200)
  └── 🏷️ Badge PEPITE (optionnel, coin haut droit)
```

### Étape 4 : Batch processing

Pour gagner du temps :

1. Générer toutes les images IA en une session
2. Créer des variantes dans Midjourney avec `/variations`
3. Utiliser "seed" pour conserver la cohérence visuelle
4. Appliquer les textes en batch avec Figma components

## 💡 Astuces d'optimisation

### Pour Midjourney

**Améliorer la qualité :**

```
--quality 2  (double le temps mais meilleure qualité)
--chaos 10   (plus de variété dans les générations)
```

**Conserver la cohérence entre images :**

```
--seed 123456  (copier le seed de l'image de référence)
```

**Variantes d'une image qui plaît :**

```
Cliquer sur "V1", "V2", "V3", "V4" pour des variantes
Ou "Make Variations" pour 4 nouvelles versions
```

### Pour DALL-E 3

**Spécifier le style :**

- "photorealistic" pour des mockups réalistes
- "modern digital illustration" pour des visuels plus graphiques
- "3D render, octane render" pour un look premium
- "flat design, minimalist" pour un style épuré

**Améliorer la qualité :**

- Demander "high quality", "professional"
- Spécifier "sharp details", "clean composition"
- Mentionner "award-winning design" pour des résultats plus soignés

**Éviter le texte généré :**

- Ajouter "no text, clean design" si vous voulez ajouter le texte après
- Ou spécifier exactement "text should say: [votre texte]"
