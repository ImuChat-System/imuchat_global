
# 🎯 Objectif du site vitrine ImuChat (version pro)

Le site doit pouvoir être montré **sans gêne** à :

* PeeL / incubateurs
* universités & collectivités
* partenaires européens **et japonais**
* premiers investisseurs / sponsors
* presse tech

👉 Ton site devient une **carte de visite stratégique**, pas une démo produit.

---

## 🧭 Arborescence recommandée (inspirée de LINE)

Je te propose une structure **simple, lisible, scalable**.

```
imuchat.app
├── / (Home)
├── /product
├── /features
├── /ai
├── /developers
├── /partners
├── /about
├── /news
├── /contact
└── /privacy
```

On détaille chaque page 👇
